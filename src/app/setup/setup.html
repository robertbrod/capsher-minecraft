Prism and Java
Install Prism Install the Prism Launcher Install Java 25 Install the the JDK 25 (LTS) via Adoptium (Temurin) During installation, ensure "Add to PATH" is checked

It is highly recommended to use the Prism Launcher and Java 25 over everything else. Not only is Prism extremely easy to use for installing/migrating, but it is highly configurable and does not require any additional Java arguments to run. Java 25 is the best because it is the most optimized and least memory intensive version that can run GTNH.

Import the Instance
Install GT:NH Download the GT:NH Prism ZIP Import the Instance Open Prism Launcher and drag the ZIP file directly into the window

** We are currently on version 2.8.4. You're client version of GT:NH MUST match the server version. **

Setup and Join!
Setup Right-click the instance > Edit > Settings > Java Set both minimum and maximum memory to 8GB Click auto-detect on Java path and select Java 25 Join the Server! Launch the instance Enter the server IP into the multiplayer screen
